Task Overview:
This project is part of the CodeAlpha Python Internship.The goal was to build a simple stock portfolio tracker that calculates the total investment value based on user input and predefined stock prices.

Features:
- Hardcoded dictionary of stock prices (e.g., AAPL, TSLA, MSFT, GOOG, AMZN)
- User inputs stock symbols and quantities
- Calculates and displays total investment value
- Optionally saves results to a `.txt` file
